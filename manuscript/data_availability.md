# Data Availability Statement

All data supporting the findings of this review are available in the study's GitHub repository. The structured data extraction spreadsheet, containing all coded variables for the 56 reviewed studies, is provided as a CSV file in the repository. No primary imaging data were generated for this study.

# Code Availability Statement

All analysis scripts and figure-generation code are publicly available at:

https://github.com/criticaldata/MRI-LMICs-survey

The repository includes all scripts necessary to reproduce the figures and statistical summaries presented in this manuscript. The analysis was conducted in Python; package versions and dependencies are specified in the repository's requirements file.
